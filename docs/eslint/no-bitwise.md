# no-bitwise

    "no-bitwise": 0,

Bitwise operations are valuable sometimes. The hinting rules
should not ban them as bitwise operators are not a source of
bugs.

[Official `no-bitwise` ESLint Rule Documentation][no-bitwise-docs]
